# API Reference

The API reference documented here details RapidREST's libraries and their public API surface as-is. For narrative, task-oriented documentation (how do I define a route, what does
`@Cache` do, ...), see the [Docs section](/docs/intro).

- **[@rapidrest/core](/docs/api/core)**: dependency injection, auto-discovery, and shared utilities.
- **[@rapidrest/service-core](/docs/api/service-core)**: the web framework itself: routing, models, auth, and more.
