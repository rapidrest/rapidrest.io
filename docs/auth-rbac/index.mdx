# Auth & RBAC

RapidREST ships authentication and role/record-level access control built in — no separate library to bolt on.

- **[Authentication](/docs/auth-rbac/authentication)** — the JWT and Basic auth strategies, requiring auth on a route, and a real login flow.
- **[Authorization & RBAC](/docs/auth-rbac/authorization-rbac)** — role checks with `@RequiresRole`, and fine-grained access control lists with `@Protect`.
