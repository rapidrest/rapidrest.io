# CLI Reference

The `@rapidrest/cli` package installs a single binary under two names — `rapidrest` and the shorter `rr` — that scaffolds projects and manages the local dev workflow.

```bash
npm install -g @rapidrest/cli
```

- **[Generate Commands](/docs/cli-reference/generate-commands)** — `generate server`, `generate model`, `generate route`, `generate job`.
- **[Add-ons](/docs/cli-reference/addons)** — `generate docker`, `generate k8s`, `generate react`, `generate react-page`.
- **[Dev Workflow](/docs/cli-reference/dev-workflow)** — `dev`, `start`, `build`.

:::note[Flags]
Every flag below is documented in its long form (`--flag-name`). A handful of commands also support a real single-character short flag, called out explicitly where it exists — don't assume every flag has one.
:::
