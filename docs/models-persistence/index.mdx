import {BaseMongoEntity, Entity, Column, DataStore} from '@rapidrest/service-core';

@DataStore('mongo')
@Entity()
export class Pet extends BaseMongoEntity {

  @Column()

}

This section will cover:

- Defining entities with `@Entity`, `@Column`, `@PrimaryColumn`, `@Index`, `@Unique`
- Choosing a datastore with `@DataStore` — MongoDB, PostgreSQL/SQLite (via TypeORM), or Redis
- The base entity classes (`BaseEntity`, `SimpleEntity`, `BaseMongoEntity`, `RecoverableBaseEntity`)
- Caching query results with `@Cache`
