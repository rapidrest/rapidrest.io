# Models & Persistence

Models are plain classes annotated with persistence decorators. The same class works whether it's backed by MongoDB, a SQL database (via TypeORM), or Redis.

- **[Entities & Columns](/docs/models-persistence/entities-columns)** - `@Entity`, `@Column`, `@PrimaryColumn`, `@Index`, `@Unique`.
- **[Datastores & Connections](/docs/models-persistence/datastores)** - `@DataStore`, connecting to MongoDB/SQL/Redis, and direct repository/connection injection.
- **[Base Entities & Caching](/docs/models-persistence/base-entities)** - the base entity classes to extend, plus `@Cache`, `@TrackChanges`, and `@ChildEntity`.
